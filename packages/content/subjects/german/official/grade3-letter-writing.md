# Letter Writing

Create a task about letter writing for {{age}}-year-old {{gender}} at difficulty level {{difficulty}}.

## Task Variations

Choose ONE of these variations:

1. **Letter Parts Identification**
   - Present a complete letter
   - Students label all parts: date, address, greeting, main text, closing, signature
   - Provide a checklist of letter elements
   - Ask which parts are missing or incorrect

2. **Writing a Complete Letter**
   - Provide a scenario (e.g., "Write to your friend about your birthday party")
   - Students write a full letter with all required parts
   - Minimum 5-6 sentences in the main text
   - Include: place and date, greeting, main text, closing, signature

3. **Choosing Appropriate Greetings**
   - Present different situations (formal vs. informal)
   - Students choose correct greetings and closings
   - Examples: *Liebe Oma*, *Hallo Max*, *Sehr geehrte Frau Müller*
   - Match greeting with situation: family, friend, teacher, unknown person

4. **Letter Content Organization**
   - Provide random sentences from a letter
   - Students organize them in logical order
   - Add missing greeting and closing
   - Ensure smooth transitions between topics

5. **Thank-You Letter Template**
   - Write a thank-you letter for a gift or visit
   - Structure: greeting → thanks → specific details → personal news → closing
   - Example: "Liebe Tante Anna, vielen Dank für..."
   - Include at least 3 specific things to mention

6. **Postcard vs. Letter**
   - Compare the format differences
   - Present a table: **Postcard** | **Letter**
   - Students list characteristics of each
   - Write one example of each (short postcard, longer letter)

7. **Letter Correction Exercise**
   - Present a letter with formatting errors
   - Errors: missing date, wrong greeting, no paragraph breaks, informal closing for formal letter
   - Students identify and correct all mistakes
   - Rewrite correctly

8. **Holiday Letter Writing**
   - Write a letter from a vacation spot
   - Include: where you are, what you're doing, weather, fun experiences
   - Use descriptive language
   - Example: "Lieber Paul, ich bin jetzt am Meer. Heute war das Wetter..."
