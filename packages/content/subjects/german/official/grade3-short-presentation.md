# Short Presentation

Create a task about short presentations for {{age}}-year-old {{gender}} at difficulty level {{difficulty}}.

## Task Variations

Choose ONE of these variations:

1. **Note Card Preparation**
   - Choose a topic (favorite book, animal, hobby, place)
   - Create 4-5 note cards with key points
   - Each card: one main point + 2-3 supporting details
   - Cards should have keywords, not full sentences

2. **Presentation Structure Planning**
   - Plan a presentation using a template
   - Sections: **Introduction** | **Main Point 1** | **Main Point 2** | **Main Point 3** | **Conclusion**
   - Example topic: "My favorite season"
   - Write brief notes for each section

3. **Poster Design**
   - Design a poster to support a presentation
   - Include: title, 3-4 main points, pictures/symbols
   - Topics: favorite sport, dream vacation, interesting animal
   - Write what you would say about each poster element

4. **Opening and Closing Phrases**
   - Practice strong presentation beginnings and endings
   - Beginnings: *Heute möchte ich euch über... erzählen.*, *Mein Thema ist...*
   - Endings: *Vielen Dank für eure Aufmerksamkeit.*, *Habt ihr Fragen?*
   - Write complete opening and closing for a chosen topic

5. **Time Management**
   - Topic for 2-minute presentation
   - Divide into time blocks: Intro (30 sec), Main (1 min), Conclusion (30 sec)
   - Write notes for each time block
   - Estimate how much to say in each part

6. **Visual Aids List**
   - Plan what to show during presentation
   - Create a table: **What to Show** | **When** | **Why**
   - Examples: real object, poster, photos, book
   - Explain how each visual helps the presentation

7. **Topic Introduction Practice**
   - Write 3 different ways to introduce the same topic
   - Make it interesting to grab attention
   - Example: "Wusstet ihr, dass...?", "Ich zeige euch heute...", "Stellt euch vor..."
   - Choose best introduction and explain why

8. **Question Preparation**
   - Prepare presentation notes on a topic
   - Anticipate 5 questions classmates might ask
   - Write answers to each question
   - Practice both presentation and Q&A
