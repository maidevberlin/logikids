# All Tenses

Create a task about all six German tenses for {{age}}-year-old {{gender}} at difficulty level {{difficulty}}.

## Task Variations

Choose ONE of these variations:

1. **Tense Overview Table**
   - Create a table with all six tenses for one verb
   - Verb example: *spielen, laufen, sehen*
   - Columns: **Tense** | **Example** | **When Used**
   - Tenses: Präsens, Präteritum, Perfekt, Plusquamperfekt, Futur I, Futur II

2. **Tense Identification**
   - Present 10-12 sentences in different tenses
   - Students identify which tense each sentence uses
   - Include all six tenses
   - Examples: "Ich gehe." (Präsens), "Ich ging." (Präteritum), "Ich bin gegangen." (Perfekt)

3. **Timeline Placement**
   - Create a timeline: **Vergangenheit** (Past) | **Gegenwart** (Present) | **Zukunft** (Future)
   - Place sentences in correct position on timeline
   - Include: "hatte gemacht" (far past), "machte" (past), "mache" (present), "werde machen" (future)

4. **Verb Conjugation Practice**
   - Choose one verb (e.g., *schreiben*)
   - Students conjugate in all six tenses for "ich"
   - Table: **Präsens** | **Präteritum** | **Perfekt** | **Plusquamperfekt** | **Futur I** | **Futur II**
   - Explain when to use each tense

5. **Tense Transformation**
   - Provide a short text (4-5 sentences) in present tense
   - Students rewrite in different tense
   - Example: Change from Präsens to Perfekt or Präteritum
   - Maintain meaning while changing time frame

6. **Completing Tense Patterns**
   - Show patterns with gaps
   - Students complete missing tenses
   - Example: "ich laufe, ich ___, ich bin gelaufen, ich ___, ich werde laufen, ___"
   - Practice with 4-5 different verbs

7. **Appropriate Tense Selection**
   - Present situations and students choose best tense
   - Situations: writing a fairy tale (Präteritum), talking about today (Perfekt), making plans (Futur I)
   - Explain why that tense fits best
   - Write example sentences for each situation

8. **Mixed Tense Story**
   - Write a short story using multiple tenses
   - Past: what happened before
   - Present: what is happening now
   - Future: what will happen next
   - Underline and label each tense used
