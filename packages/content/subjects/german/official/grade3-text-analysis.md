---
id: grade3-text-analysis
name: Text Analysis and Literary Understanding
description: Analyzing plot sequences, describing characters, explaining figurative language, and comparing different text types
grade: 3
focus: Literary comprehension and text structure analysis
difficulty: medium
learning_objectives:
  - Analyze and visualize plot sequences and key story events
  - Describe character traits, appearance, and relationships from multiple perspectives
  - Explain figurative language and literary devices in age-appropriate texts
  - Compare and distinguish between narrative, informational, and functional text types
  - Rewrite content across different genres or media forms
  - Identify narrative structure elements (situation, event, resolution)
  - Read poetry rhythmically to understand sound patterns and rhythm
problem_types:
  - Plot sequence ordering
  - Character trait identification
  - Figurative language explanation
  - Text type comparison
  - Story event mapping
  - Genre transformation
  - Perspective analysis
  - Narrative structure identification
difficulty_guidelines:
  easy:
    - Use texts with clear 3-part structure and obvious character traits
    - Figurative language limited to simple comparisons and common metaphors
    - Text types have distinct, easily recognizable features
  medium:
    - Use texts with multiple plot points requiring event prioritization
    - Include mixed character traits requiring evidence from different passages
    - Figurative language includes personification and symbolic elements
  hard:
    - Use texts with parallel plots or flashback sequences
    - Character analysis requires inference from dialogue and actions
    - Compare subtle differences between similar text types or hybrid forms
prerequisites:
  - grade2-reading-simple-texts
  - grade2-retelling-stories
real_world_context:
  - Analyzing characters in favorite books and discussing their motivations
  - Understanding the difference between a recipe (functional), an animal report (informational), and a fairy tale (narrative)
  - Explaining what authors mean by phrases like "as brave as a lion" or "time crawled by"
  - Creating story maps or timelines for chapter books and longer narratives
  - Transforming written stories into comic strips or dramatic performances
anti_patterns:
  - Avoid texts requiring abstract symbolism beyond concrete imagery
  - Don't use cultural references unfamiliar to 8-9 year olds
  - Avoid character psychology requiring adult emotional understanding
  - Don't ask for text type distinctions based on formatting alone
  - Avoid figurative language that depends on idiomatic cultural knowledge
version: 1
version_notes: Initial version based on LehrplanPLUS Grade 3 reading competencies (2.1, 2.4)
