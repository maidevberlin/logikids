# Sentence Types

Create a task about sentence types (main and subordinate clauses) for {{age}}-year-old {{gender}} at difficulty level {{difficulty}}.

## Task Variations

Choose ONE of these variations:

1. **Main vs. Subordinate Clause**
   - Present complex sentences
   - Students identify and label **Hauptsatz** (main clause) and **Nebensatz** (subordinate clause)
   - Examples: "Ich bleibe zu Hause, weil es regnet.", "Sie sagt, dass sie kommt."
   - Underline each part with different colors

2. **Subordinating Conjunction Recognition**
   - List common subordinating conjunctions
   - Conjunctions: *weil, dass, wenn, obwohl, als, während, damit*
   - Students identify which conjunctions introduce subordinate clauses
   - Write example sentences with each

3. **Comma Placement Practice**
   - Provide sentences without commas
   - Students add commas before subordinate clauses
   - Rule: comma before subordinate clause
   - Examples: "Ich gehe nach Hause [,] weil ich müde bin.", "Er sagt [,] dass er kommt."

4. **Sentence Combining**
   - Provide two simple sentences
   - Students combine using subordinating conjunctions
   - Example: "Es regnet. Ich bleibe zu Hause." → "Ich bleibe zu Hause, weil es regnet."
   - Practice with 5-6 sentence pairs

5. **Subordinate Clause Word Order**
   - Focus on verb position in subordinate clauses
   - Rule: verb goes to the end in subordinate clauses
   - Examples: "Ich weiß, dass **du kommst**." (not "du **kommst**")
   - Students correct sentences with wrong word order

6. **Types of Subordinate Clauses**
   - Categorize subordinate clauses by function
   - Types: **Begründung** (weil), **Aussage** (dass), **Zeit** (als, wenn), **Bedingung** (wenn)
   - Create a table sorting example sentences
   - Write one sentence for each type

7. **Building Complex Sentences**
   - Start with a main clause
   - Add subordinate clause using given conjunction
   - Main: "Ich freue mich" + *dass* → "Ich freue mich, dass du kommst."
   - Practice with different conjunctions and main clauses

8. **Error Detection**
   - Present sentences with errors
   - Errors: missing comma, wrong word order, wrong conjunction
   - Students identify and correct mistakes
   - Examples: "Ich weiß dass du kommst." (missing comma), "Ich gehe weil nach Hause ich müde bin." (wrong order)
