---
id: grade5-media-literacy
name: Media Literacy
description: Understanding library systems, digital research, and reflecting on media use
grade: 5
focus: Information literacy and critical media reflection
difficulty: easy
learning_objectives:
  - Understand library organization systems (alphabetical and thematic ordering)
  - Use age-appropriate search engines effectively for information gathering
  - Evaluate basic credibility of online sources
  - Reflect on personal media consumption patterns (duration, effects, purposes)
  - Analyze basic audio and film techniques (perspective, music, sound effects)
  - Compare how literary texts are presented across different media (book, film, audio)
  - Distinguish fact from opinion in media content
problem_types:
  - Library navigation
  - Search strategy
  - Source evaluation
  - Media reflection
  - Technique analysis
  - Cross-media comparison
  - Information verification
difficulty_guidelines:
  easy:
    - Single library classification task (alphabetical or thematic)
    - Basic search engine query formulation for familiar topics
    - Simple credibility checks (author, date, website type)
  medium:
    - Multiple organization systems or complex library tasks
    - Refined search strategies with multiple keywords
    - Compare credibility of 2-3 sources on same topic
  hard:
    - Complex research tasks combining multiple library resources
    - Evaluate search results and adjust strategy based on findings
    - Analyze media techniques and their persuasive effects
prerequisites: []
real_world_context:
  - Using school or public library to find books on topics of interest
  - Searching online for information about homework topics or personal interests
  - Deciding which websites are trustworthy when researching a school project
  - Reflecting on daily screen time and choosing which media to consume
  - Comparing a book you read with its film adaptation
  - Navigating library catalog systems to locate specific books for book reports
  - Evaluating YouTube educational videos for accuracy when learning about science topics
  - Discussing with parents about appropriate media choices for entertainment and learning
  - Analyzing how music and sound effects create suspense in audio books or radio plays
  - Comparing illustrated book versions with graphic novel adaptations of the same story
  - Identifying sponsored content versus objective information on children's websites
  - Using search engines to find reliable sources for geography or history presentations
anti_patterns:
  - Avoid requiring knowledge of advanced library classification systems
  - Don't assume familiarity with specific search engines or interfaces
  - Avoid abstract media theory beyond concrete observation
  - Don't require technical terminology for film or audio production
  - Avoid topics requiring adult-level source evaluation skills
version: 2
version_notes: Expanded real_world_context from 5 to 12 items for better variety
