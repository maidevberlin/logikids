# Spelling - Word Families

Create a task about word families for {{age}}-year-old {{gender}} at difficulty level {{difficulty}}.

## Task Variations

Choose ONE of these variations:

1. **Finding Related Words**
   - Present a base word (e.g., *fahren, schreiben, laufen*)
   - Students list 6-8 related words from the same family
   - Include different word forms: nouns, verbs, adjectives
   - Examples: *fahren* → *Fahrt, Fahrer, Abfahrt, verfahren, fahrbar*

2. **Identifying the Base Word**
   - Provide a group of related words
   - Students identify the common base word
   - Examples: *Läufer, Lauf, verlaufen, gelaufen* → base: *laufen*
   - Include 3-4 different word families

3. **Word Family Trees**
   - Start with a base word in the center
   - Students create branches with related words
   - Organize by word type: nouns, verbs, adjectives
   - Example tree: *schreib-* (write) with branches *Schreiber, schreiben, Schrift, beschreiben*

4. **Spelling from Base Word**
   - Present difficult spellings within word families
   - Students use base word to derive correct spelling
   - Examples: *fährst* (from *fahren*), *läuft* (from *laufen*)
   - Explain how the base word helps with spelling

5. **Sorting Exercise**
   - Provide 12-15 mixed words from 3 different families
   - Students sort them into a table by word family
   - Example families: *gehen, sehen, stehen*
   - Include tricky words like *Ansehen, Gestehen, Vergehen*

6. **Missing Word Completion**
   - Show sentences with blanks for word family members
   - Students complete using the correct form from the given base word
   - Example: Base *fahren*: "Die ___ zum Meer dauert lange." (*Fahrt*)

7. **Word Family Expansion**
   - Start with 2-3 words from a family
   - Students add 5 more related words
   - Include prefixes and suffixes: *ver-, ab-, -ung, -er*
   - Example: *schreiben, Schrift* → add *beschreiben, Schreibung, verschreiben*

8. **Compound Word Creation**
   - Use word family members to create compound words
   - Example: *Fahrt* → *Autofahrt, Bergfahrt, Schifffahrt*
   - Students create 5-6 meaningful compounds
   - Explain the meaning of each compound
