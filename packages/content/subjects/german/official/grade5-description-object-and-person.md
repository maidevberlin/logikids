# Description - Object and Person

Create {{count}} varied tasks about **describing objects and persons precisely** for a {{age}}-year-old student. Focus on {{focus_area}}.

## Task Variations

### Variation 1: Object Description - Structure
Students learn systematic object description from general to specific.

**Example:**
"Describe your school bag using this structure:

1. **General category:** What type of object is it?
   *'Mein Schulranzen ist...'*

2. **Size and shape:** How big? What shape?
   *'Er ist etwa 40 cm hoch und hat eine rechteckige Form...'*

3. **Material and color:** What is it made of? What color?
   *'Er besteht aus wasserabweisendem Stoff und ist blau mit grünen Streifen...'*

4. **Special features:** What makes it unique?
   *'Auf der Vorderseite ist ein reflektierender Aufkleber mit einem Dinosaurier...'*

5. **Function/Use:** What is it for? Why is it important?
   *'Ich benutze ihn täglich für meine Schulbücher und...'*"

### Variation 2: Person Description - Appearance
Students describe physical appearance systematically.

**Example:**
"Write a precise description of a person for a missing person poster:

**Structure (from top to bottom):**
1. **Overall impression:** Age, height, build
   *'Die Person ist etwa 12 Jahre alt, 1,50 m groß, schlank...'*

2. **Face:** Shape, complexion, special features
   *'Ovales Gesicht, helle Haut, Sommersprossen auf der Nase...'*

3. **Hair:** Color, length, style
   *'Dunkelblonde, schulterlange Haare, meist zu einem Pferdeschwanz gebunden...'*

4. **Eyes:** Color, shape
   *'Blaue Augen, mandelförmig...'*

5. **Clothing:** What was the person wearing?
   *'Zuletzt gesehen mit roter Jacke, blauen Jeans, weißen Turnschuhen...'*

6. **Distinguishing marks:** Scars, birthmarks, etc.
   *'Kleines Muttermal am linken Handgelenk...'*"

### Variation 3: Person Description - Character
Students describe personality traits with supporting examples.

**Example:**
"Describe a person's character, not just appearance:

**Don't just list adjectives - give examples!**

❌ *'Meine Freundin ist nett, lustig und hilfsbereit.'*
✓ *'Meine Freundin Lisa ist sehr hilfsbereit. Letzte Woche hat sie mir bei den Hausaufgaben geholfen, obwohl sie selbst viel zu tun hatte. Sie bringt mich oft zum Lachen mit ihren lustigen Geschichten...'*

**Describe a friend or family member:**
- Choose 3 main character traits
- For each trait, give a concrete example
- Explain why this trait is important to you"

### Variation 4: Combining Appearance and Character
Students connect outer appearance with inner qualities.

**Example:**
"Describe a book character (like Harry Potter, Pippi Longstocking, etc.):

**Part 1 - Appearance:**
*'Harry Potter ist ein etwa 11-jähriger Junge mit schwarzen, zerzausten Haaren und grünen Augen. Er trägt eine runde Brille und hat eine blitzförmige Narbe auf der Stirn...'*

**Part 2 - Character:**
*'Obwohl Harry äußerlich eher unauffällig wirkt, ist er sehr mutig und loyal. Seine Narbe erinnert an...'*

**Connection:** How does appearance reflect or contrast with personality?
*'Harrys zerzauste Haare und die alte Brille zeigen, dass er sich nicht viel aus Äußerlichkeiten macht...'*"

### Variation 5: Sensory Description
Students use all senses to describe an object.

**Example:**
"Describe a favorite food using all senses:

1. **Sight (Aussehen):** What does it look like?
   *Color, shape, size*

2. **Touch (Gefühl):** How does it feel?
   *Smooth, rough, soft, hard, warm, cold*

3. **Smell (Geruch):** How does it smell?
   *Fresh, sweet, spicy, strong*

4. **Taste (Geschmack):** How does it taste?
   *Sweet, sour, salty, bitter, spicy*

5. **Hearing (Geräusch):** Does it make a sound?
   *Crunchy, crackling, sizzling*

**Example - Pizza:** *'Die Pizza ist rund und goldbraun. Der geschmolzene Käse glänzt. Wenn man sie berührt, ist sie heiß und etwas fettig. Sie duftet nach Tomaten, Käse und Oregano. Beim Hineinbeißen knackt der Rand...'*"

### Variation 6: Comparison Description
Students describe by comparing to something familiar.

**Example:**
"Describe an unusual object by comparing it to known things:

**Object:** A pineapple (for someone who has never seen one)

**Use comparisons:**
- **Size:** 'So groß wie eine große Faust oder ein kleiner Fußball'
- **Shape:** 'Oval, ähnlich wie ein Rugby-Ball, aber dicker'
- **Surface:** 'Die Schale hat ein Muster wie Fischschuppen, aber härter'
- **Color:** 'Gelbbraun, wie getrocknetes Gras'
- **Top:** 'Oben wachsen steife, grüne Blätter wie eine Krone'

**Your turn:** Describe a strange object (like a cactus, artichoke, or dragon fruit) using comparisons."

### Variation 7: Lost and Found Description
Students write precise descriptions for practical purposes.

**Example:**
"Write a 'Lost Item' notice for a bulletin board:

**Lost: Bicycle**

❌ **Too vague:** *'Fahrrad verloren. Blau. Bei Fund bitte melden.'*

✓ **Precise:**
*'**Verloren:** Mountainbike
**Marke:** KTM
**Farbe:** Königsblau mit weißen Streifen
**Größe:** 24 Zoll (für Kinder)
**Besonderheiten:**
- Aufkleber vom FC Bayern am Lenker
- Kleine Delle im Schutzblech vorne rechts
- Fahrradklingel mit Smiley-Gesicht
**Verloren am:** 15. Oktober, Nähe Stadtpark
**Kontakt:** Tel. 0123-456789'*

**Your turn:** Write a lost item notice for something valuable."

### Variation 8: Before-and-After Description
Students describe changes in objects or persons over time.

**Example:**
"Describe how a person or object has changed:

**Example - A tree through seasons:**

**Spring (Frühjahr):**
*'Im Frühjahr ist der Apfelbaum voller weißer und rosa Blüten. Seine Äste biegen sich unter dem Gewicht der vielen Knospen...'*

**Summer (Sommer):**
*'Im Sommer ist der Baum dicht belaubt. Kleine grüne Äpfel hängen zwischen den dunkelgrünen Blättern...'*

**Autumn (Herbst):**
*'Im Herbst leuchtet der Baum mit roten, reifen Äpfeln. Die Blätter färben sich gelb und orange...'*

**Winter (Winter):**
*'Im Winter steht der Baum kahl da. Seine schwarzen Äste ragen in den grauen Himmel...'*

**Your turn:** Describe how a person or place has changed (a grandparent, your room, your hometown)."
