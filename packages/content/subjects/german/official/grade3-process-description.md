# Process Description

Create a task about process descriptions for {{age}}-year-old {{gender}} at difficulty level {{difficulty}}.

## Task Variations

Choose ONE of these variations:

1. **Step-by-Step Instructions**
   - Provide a simple process (making a sandwich, brushing teeth, watering plants)
   - Students write 5-7 numbered steps
   - Must use sequence words: *zuerst, dann, danach, anschließend, zum Schluss*
   - Each step should be one clear sentence

2. **Scrambled Steps**
   - Present process steps in wrong order
   - Students number them correctly (1-6)
   - Add appropriate sequence words to each step
   - Example: Making hot chocolate, planting a flower, cleaning a room

3. **Missing Sequence Words**
   - Provide a process description without sequence words
   - Students add appropriate connecting words
   - Blanks: "___ schneidest du das Brot. ___ streichst du Butter darauf."
   - Word bank: *Zuerst, Dann, Danach, Zum Schluss*

4. **Visual to Written Instructions**
   - Describe 4-5 pictures showing a process
   - Students write step-by-step instructions
   - Examples: making a paper airplane, preparing a salad, setting a table
   - Include at least 5 steps with sequence words

5. **Game Instructions**
   - Write instructions for a simple game
   - Include: what you need, how to play (4-6 steps), how to win
   - Use clear sequence words and numbered steps
   - Example games: hide and seek, tag, board game

6. **Recipe Writing**
   - Write a simple recipe (fruit salad, sandwich, smoothie)
   - Two sections: **Ingredients** (list) and **Instructions** (numbered steps)
   - Use imperative form: "Schneide...", "Mische...", "Gib..."
   - Include at least 5 preparation steps

7. **Error Detection**
   - Present a process description with logical errors
   - Steps are out of order or sequence words are wrong
   - Students identify and correct the mistakes
   - Example: "Zuerst isst du den Kuchen. Dann backst du ihn."

8. **Comparison of Sequence Words**
   - Create a table: **Sequence Word** | **Meaning** | **Use in Sentence**
   - Words: *zuerst, dann, danach, anschließend, zum Schluss, schließlich*
   - Write one process using at least 4 different sequence words
   - Explain when to use each word
