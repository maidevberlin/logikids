---
id: grade11-wissenschaftliches-schreiben
name: Academic Writing
description: Scholarly writing conventions including citations, footnotes, bibliography, and academic register
grade: 11
focus: Scholarly writing methodology and conventions
difficulty: hard
learning_objectives:
  - Apply academic citation conventions consistently across different source types
  - Integrate and attribute sources properly using footnotes and in-text citations
  - Develop scholarly arguments with appropriate evidence and methodology
  - Use formal academic register and discipline-specific terminology
  - Format papers according to academic standards and style guides
  - Distinguish between primary and secondary sources in scholarly contexts
  - Create properly formatted bibliographies following academic conventions
problem_types:
  - Citation formatting
  - Footnote creation
  - Bibliography construction
  - Source attribution
  - Register analysis
  - Academic paraphrasing
  - Reference verification
  - Style guide application
  - Quotation integration
  - Methodology description
difficulty_guidelines:
  easy:
    - Format citations for standard sources (books, articles)
    - Apply single citation style consistently
    - Use appropriate academic vocabulary in simple contexts
  medium:
    - Handle complex source types (edited volumes, online sources, archives)
    - Switch between citation styles appropriately
    - Maintain formal register throughout extended academic texts
  hard:
    - Resolve citation edge cases and ambiguous sources
    - Integrate multiple citation systems (footnotes + bibliography)
    - Apply discipline-specific conventions and advanced academic discourse
real_world_context:
  - Writing a Facharbeit on Romantic literature for your German class
  - Preparing a research paper on climate policy for social studies
  - Creating an annotated bibliography for a history seminar on the Cold War
  - Writing a literary analysis of Goethe's Faust with scholarly sources
  - Researching Enlightenment philosophy using primary and secondary texts
  - Preparing a scientific paper on genetic engineering with journal citations
  - Writing about architectural history using museum archives and catalogs
  - Creating a research proposal for a psychology study with literature review
  - Analyzing historical speeches using archival documents and commentaries
  - Writing about economic theory using academic journals and textbooks
  - Researching medieval manuscripts for a paleography seminar
  - Preparing a comparative analysis of political systems with policy documents
  - Writing about film theory using scholarly monographs and criticism
  - Investigating linguistic change through corpus data and research articles
  - Creating a thesis chapter on environmental ethics with philosophical sources
anti_patterns:
  - Don't use informal language or contractions in academic writing
  - Avoid citing Wikipedia or unreliable web sources as scholarly evidence
  - Don't mix citation styles (e.g., MLA and Chicago) in the same paper
  - Avoid footnotes that merely repeat information already in the text
  - Don't include sources in the bibliography that aren't cited in the paper
