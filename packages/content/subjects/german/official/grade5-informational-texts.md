**Create a task where {{student_name}} analyzes and works with informational (non-fiction) texts.**

Your task should:
- Focus on one of these variations:
  1. **Text Structure**: Identify and analyze the structure of an informational text (introduction, main sections, conclusion, headings, paragraphs)
  2. **Facts vs. Opinions**: Distinguish between factual statements and opinion statements in a text
  3. **Information Extraction**: Find and organize specific information from an informational text
  4. **Argument Evaluation**: Identify the author's main argument and evaluate the supporting evidence
  5. **Text Purpose**: Analyze why the text was written (to inform, to persuade, to explain, to instruct)
  6. **Source Comparison**: Compare how two different sources present the same topic
  7. **Visual Elements**: Analyze how diagrams, tables, photos, or graphics support the text
  8. **Writing Practice**: Write your own informational text on a given topic using appropriate structure

- Use age-appropriate topics (animals, technology, history, sports, environment, culture)
- Provide text excerpts of 150-300 words when analyzing
- Include examples of both facts and opinions
- Show clear text structure with paragraphs and/or headings
- Use tables to organize information or comparisons

**Important**:
- Ensure diverse, engaging topics each time (avoid repetition)
- Vary the text type (encyclopedia entry, magazine article, instructional text, news report)
- Change the analytical focus (structure, content, purpose, credibility)
- Provide fresh examples of facts, opinions, and arguments
