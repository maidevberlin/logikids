---
id: grade3-composition-planning
name: Composition and Planning
description: Learn to plan and organize ideas for different types of writing using mind maps, word lists, and narrative structures
grade: 3
focus: Writing planning and organization strategies
difficulty: medium
learning_objectives:
  - Collect and organize ideas using mind maps and word lists
  - Extract and apply text features from model texts
  - Plan narratives with situation-event-resolution structure
  - Structure informational texts with logical sequencing
  - Develop argumentative texts with positions, reasons, and examples
  - Build personal word banks for different writing purposes
  - Use writing for text comprehension through perspective-taking
problem_types:
  - Mind map creation
  - Story structure planning
  - Argument organization
  - Word list building
  - Text feature identification
  - Informational text outlining
  - Perspective-taking writing
  - Note-taking for comprehension
difficulty_guidelines:
  easy:
    - Simple three-part narrative plans (who, what happened, how it ended)
    - Mind maps with 3-5 main branches for familiar topics
    - Word lists with 5-8 related words from everyday contexts
  medium:
    - Detailed narrative plans with setting, characters, and emotions
    - Informational text outlines with 3-4 main points and supporting details
    - Mind maps integrating connections between ideas
  hard:
    - Argumentative text plans with claim, 2-3 reasons, and counterargument
    - Multi-layered mind maps showing relationships and subcategories
    - Perspective-switching plans requiring character voice adaptation
prerequisites:
  - grade2-writing-simple-sentences
  - grade2-retelling-stories
real_world_context:
  - Planning a story about a school trip before writing it in a journal
  - Creating a mind map to organize ideas for a class presentation on animals
  - Making an argument plan with reasons why the class should visit a museum
  - Building a word bank of descriptive adjectives before writing about seasons
  - Outlining steps in logical order before writing a recipe or instructions
anti_patterns:
  - Avoid requiring more than 3-4 main organizational levels in mind maps
  - Don't expect fully formed sentences in planning stages
  - Avoid abstract topics unrelated to 8-9 year old experiences
  - Don't combine multiple planning techniques in a single task
  - Avoid planning prompts without clear purpose or text type guidance
version: 1
version_notes: Initial concept for German Grade 3 composition planning aligned with LehrplanPLUS Bayern curriculum area 3.2
