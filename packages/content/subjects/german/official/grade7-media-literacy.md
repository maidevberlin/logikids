---
id: grade7-media-literacy
name: Media Literacy and Source Evaluation
description: Internet research, evaluating source reliability, comparing information quality
grade: 7
focus: Critical evaluation of digital information sources
difficulty: medium
learning_objectives:
  - Conduct effective internet research using strategic search terms
  - Assess search results by comparing information quality across platforms
  - Evaluate information quality and argumentative validity across sources
  - Identify reliable versus unreliable online sources using multiple criteria
  - Understand the difference between opinion pieces and factual reporting
  - Cross-reference information from multiple sources to verify accuracy
  - Reflect on responsible media consumption and digital citizenship
problem_types:
  - Search strategy formulation
  - Source reliability assessment
  - Information quality comparison
  - Fact versus opinion distinction
  - Cross-source verification
  - Bias identification
  - Filter bubble awareness
  - Argumentative validity analysis
difficulty_guidelines:
  easy:
    - Evaluate single source using explicit credibility markers (author, date, domain)
    - Distinguish clear fact statements from obvious opinion expressions
    - Formulate basic search queries for straightforward topics
  medium:
    - Compare 2-3 sources and identify quality differences
    - Recognize subtle bias or persuasive language in texts
    - Develop refined search strategies using multiple keywords and operators
  hard:
    - Evaluate argumentative validity across conflicting sources
    - Analyze how algorithmic filtering affects information exposure
    - Synthesize information from multiple sources while assessing credibility
prerequisites:
  - grade6-informational-texts
  - grade5-media-literacy
real_world_context:
  - Researching current events online and distinguishing news reporting from opinion blogs
  - Comparing different news websites' coverage of the same event to identify bias
  - Verifying viral social media claims by cross-referencing with reliable sources
  - Recognizing echo chambers and filter bubbles in personalized social media feeds
  - Using fact-checking websites to verify questionable claims encountered online
  - Evaluating Wikipedia articles by checking sources and comparing to authoritative references
  - Researching topics for school projects while assessing which websites provide reliable information
  - Analyzing how influencers present sponsored content versus genuine recommendations
  - Identifying clickbait headlines and misleading thumbnails in online content
  - Comparing coverage of scientific topics across mainstream media and specialized sources
  - Evaluating credibility of health or environmental claims shared on social media
  - Recognizing how search algorithms personalize results and affect information access
anti_patterns:
  - Avoid requiring technical knowledge of SEO or search algorithms
  - Don't assume access to specific platforms or paywalled sources
  - Avoid politically controversial topics requiring mature judgment
  - Don't ask source evaluation without providing evaluation criteria
  - Avoid abstract media theory beyond concrete observation and analysis
version: 1
version_notes: Initial creation for Grade 7 German curriculum
