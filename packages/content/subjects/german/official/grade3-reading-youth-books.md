# Reading Youth Books

Create a task about reading youth books for {{age}}-year-old {{gender}} at difficulty level {{difficulty}}.

## Task Variations

Choose ONE of these variations:

1. **Reading Journal Entry**
   - Create a template for journal entries
   - Sections: **Date**, **Pages Read**, **Main Events**, **Favorite Part**, **Questions**
   - Students fill in for their current book
   - Include predictions about what happens next

2. **Character Description**
   - Choose a main character from a youth book
   - Create a character profile
   - Include: **Name**, **Age**, **Appearance**, **Personality**, **What I Like About Them**
   - Draw or describe the character

3. **Book Recommendation**
   - Write a recommendation for a favorite book
   - Include: title, author, genre, summary (3-4 sentences), why others should read it
   - Rate the book: excitement, humor, suspense
   - "I recommend this book because..."

4. **Chapter Summary Practice**
   - Summarize one chapter in 3-5 sentences
   - Include: main events, important characters, problems or questions
   - Practice finding key information
   - Compare summary with actual chapter length

5. **Book Poster Design**
   - Design a poster for a youth book
   - Include: title, author, main character, exciting scene, rating
   - Add a catchy phrase: "If you like adventure, read..."
   - Plan layout and content

6. **Comparing Two Books**
   - Read two books and compare
   - Table: **Book 1** | **Book 2** | for categories: **Main Character**, **Setting**, **Problem**, **My Favorite**
   - Which book did you prefer and why?

7. **Reading Goal Setting**
   - Set monthly reading goals
   - Table: **Month** | **Books to Read** | **Pages Goal** | **Genres to Try**
   - Track progress with checkmarks
   - Reflect: Did I reach my goal? What helped/hindered?

8. **Book Discussion Questions**
   - Create 5-6 discussion questions about a book
   - Questions should be open-ended (not yes/no)
   - Examples: "Why did the character...?", "What would you do if...?", "How did the character change?"
   - Practice asking questions that spark conversation
