name: Linear Functions
description: Understanding and working with linear functions y = mx + t, including graphing, interpreting slope and y-intercept, finding zeros and intersection points, and solving linear inequalities.
grade: 8
focus: Linear relationships and algebraic representation
difficulty: medium
learning_objectives:
  - Interpret and graph linear functions in the form y = mx + t
  - Determine slope (m) and y-intercept (t) from graphs and tables
  - Find zeros and intersection points of linear functions
  - Solve linear inequalities and express solutions in interval notation
  - Set up linear equations from real-world situations
  - Recognize direct proportionality as a special case where t = 0
  - Analyze how parameter changes affect linear function graphs
problem_types:
  - Graphing from equation
  - Finding slope and intercept
  - Determining equation from graph
  - Calculating zeros
  - Intersection points
  - Linear inequalities
  - Real-world modeling
  - Parameter interpretation
  - Direct proportionality
  - Parallel and perpendicular lines
difficulty_guidelines:
  easy:
    - Use integer values for m and t (slope and y-intercept)
    - Simple grids with clear scale (e.g., increments of 1 or 2)
    - Positive slopes for graphing tasks
  medium:
    - Include fractional or decimal values for m and t
    - Negative slopes and y-intercepts
    - Solve inequalities requiring algebraic manipulation
  hard:
    - Multiple-step problems combining several concepts (e.g., find equation, then solve inequality)
    - Complex real-world scenarios requiring equation setup
    - Finding equations from two points or one point and slope
