age_guidelines:
  10:
    - Use clear visual representations with distinct shapes
    - Focus on basic shape identification and simple properties
    - Introduce symmetry concepts with visual aids
    - Present angles in standard orientations initially

difficulty_guidelines:
  easy:
    - Direct identification of common shapes (triangles, rectangles, squares)
    - Single-property recognition (e.g., "has right angles")
    - Counting sides, vertices, or symmetry lines
    - Basic angle type classification (acute, right, obtuse)
    - Identifying symmetry when axis is shown
    - Simple tool selection for constructions
  medium:
    - Multi-property classification (e.g., "equal sides AND right angles")
    - Distinguishing between similar shapes (rectangle vs. parallelogram)
    - Determining number of symmetry lines without visual aids
    - Angle relationships (complementary, supplementary)
    - Step-by-step construction reasoning
    - Shape comparisons and shared properties
  hard:
    - Hierarchical classification (square is a special rectangle)
    - Complex symmetry analysis (point symmetry orders)
    - Reflex angle identification and measurement
    - Error analysis in constructions or classifications
    - Composite figures with multiple shapes
    - Inverse reasoning ("which is NOT..." questions)
