# Digital Tools - Basic Use

Create tasks that teach students to use digital tools effectively for language learning and information gathering.

**Task Requirements:**
- Focus on practical, age-appropriate digital skills
- Teach online dictionary use (English-English and bilingual)
- Practice with learning platforms and educational apps
- Introduce basic digital presentation skills
- Develop critical evaluation of online sources
- Emphasize digital safety and reliability
- Make tasks hands-on and interactive
- Age-appropriate for 10-11 year olds

**Generate varied task types:**

## Variation 1: Online Dictionary Practice
Learn to use online English dictionaries effectively.

**Example:**

**Task: Dictionary Treasure Hunt**

Use an online dictionary (like Cambridge Learner's Dictionary or Oxford Learner's Dictionary) to find:

1. Look up the word *{{word}}*:
   - Pronunciation: _____ (copy the phonetic spelling)
   - Definition: _____
   - Example sentence: _____
   - Word type (noun/verb/adjective): _____

2. Find {{number}} synonyms (words with similar meaning) for *happy*:
   - _____
   - _____
   - _____

3. Use the dictionary's example sentences for *run*. Write the sentence that matches "running for exercise":
   _____

4. Challenge: Look up a word you don't know from today's lesson. Write what you learned.

**Useful features to teach:**
- Audio pronunciation buttons
- Example sentences
- Word families (happy → happiness, unhappy)
- Picture dictionaries

## Variation 2: Learning Platform Navigation
Practice using digital learning platforms.

**Example:**

**Platform Skills Checklist**

Using {{platform name}}, complete these tasks:

☐ Log in to your account
☐ Find the {{topic}} lesson
☐ Complete {{number}} exercises
☐ Check your score
☐ Review your mistakes
☐ Mark difficult words with a star
☐ Log out properly

**Reflection:**
- What was easy? _____
- What was difficult? _____
- What did you learn? _____
- How many exercises did you complete correctly? ___/___

## Variation 3: Digital Presentation Creation
Create a simple digital poster or slide presentation.

**Example:**

**Make a Digital Poster: "My Favorite Animal"**

Use PowerPoint / Google Slides / Canva to create ONE slide with:

**Must include:**
1. Title: "My Favorite Animal: {{animal}}"
2. One picture of the animal
3. {{number}} facts about the animal:
   - Where it lives: _____
   - What it eats: _____
   - Interesting fact: _____

4. Your name at the bottom

**Design tips:**
- Choose a background color
- Use big, clear font
- Don't add too many pictures
- Check spelling!

**Save your work** and share with the class.

## Variation 4: Source Evaluation Activity
Learn to evaluate if a website is good for learning.

**Example:**

**Website Detective**

Look at these two websites about {{topic}}. Which one is better for learning English?

**Website A:** {{description}}
**Website B:** {{description}}

**Check these things:**

| Feature | Website A | Website B |
|---------|-----------|-----------|
| Easy to read | Yes/No | Yes/No |
| Has pictures | Yes/No | Yes/No |
| Information is clear | Yes/No | Yes/No |
| Has ads or pop-ups | Yes/No | Yes/No |
| Good for my age | Yes/No | Yes/No |
| I can trust it | Yes/No | Yes/No |

**My choice:** I think Website ___ is better because _____

**Red flags to watch for:**
- Too many advertisements
- Spelling mistakes
- No author information
- Looks unprofessional

## Variation 5: Bilingual Dictionary Comparison
Compare English-English and English-German dictionaries.

**Example:**

Look up the word *{{word}}* in BOTH dictionaries:

**English-German Dictionary:**
- German translation: _____
- Easy to understand? Yes/No

**English-English Dictionary (Learner's):**
- English definition: _____
- Example sentence: _____
- Easy to understand? Yes/No

**Questions:**
1. Which dictionary is better for learning English? Why?
2. When would you use an English-German dictionary?
3. When would you use an English-English dictionary?

**Answer:** Both are useful! Use English-German when you need a quick translation. Use English-English to really understand how to use the word.

## Variation 6: Video Learning Activity
Use educational videos effectively for learning.

**Example:**

**Watch and Learn: How to Use YouTube for English Learning**

Watch the video about {{topic}} (2 minutes).

**Before watching:**
- What do you want to learn? _____

**While watching:**
- Pause and write {{number}} new words: _____, _____, _____
- What pictures help you understand? _____

**After watching:**
- What did you learn? _____
- Watch again if needed!
- Try the automatic subtitles (CC button)

**Tips for video learning:**
- You can pause, rewind, watch again
- Slow down the speed if too fast (settings ⚙)
- Turn on English subtitles to read along
- Take notes!

## Variation 7: Online Quiz Platform Practice
Complete a quiz on a learning platform.

**Example:**

**Quizlet/Kahoot Practice**

Topic: {{vocabulary set}}

**Tasks:**
1. Find the {{topic}} quiz
2. Choose "Learn" mode first (not test!)
3. Go through all flashcards
4. Try the "Match" game
5. When ready, take the "Test"
6. Record your score: ___/___

**Learning strategies:**
- Star (⭐) difficult words to review later
- Use the audio pronunciation
- Try "Write" mode to practice spelling
- Repeat until you get 100%!

## Variation 8: Digital Safety and Etiquette
Learn about using English safely online.

**Example:**

**Online Safety Rules**

True or False?

1. I should use my real full name in online games. (False - use a nickname)
2. If I don't understand something, I can ask my teacher or parents. (True)
3. It's okay to share my email with everyone online. (False - only with people you trust)
4. I should be polite when writing comments in English. (True)
5. Free online dictionaries are safe to use. (True - but with adult guidance)

**Digital manners in English:**
- Say "please" and "thank you"
- Don't write in ALL CAPS (it's like shouting!)
- Be kind in comments
- Ask before sharing others' work

## Variation 9: Search Engine Skills
Learn to search for English learning resources.

**Example:**

**Google Search Practice**

Learn to find good English learning resources:

**Task 1: Search for images**
Search: "{{topic}} pictures for kids"
- How many results? _____
- Choose {{number}} good pictures for learning
- Why are they useful? _____

**Task 2: Search for information**
Search: "fun facts about {{topic}} for children"
- Find {{number}} interesting facts
- Copy them in your own words (no copy-paste!)

**Search tips:**
- Use simple words
- Add "for kids" or "for children"
- Add "English learning" if needed
- Try different words if you don't find what you want

**Safe Search:** Make sure Safe Search is ON (ask your teacher!)

## Variation 10: Creating a Word Document
Learn basic word processing for English writing.

**Example:**

**Write and Format a Letter**

Use Microsoft Word / Google Docs to write a letter to your pen pal:

**Tasks:**
1. Open a new document
2. Choose font: {{recommended font}} (easy to read)
3. Font size: {{size}}
4. Write your letter ({{min}} sentences):
   - Greeting: Dear {{name}},
   - Ask {{number}} questions
   - Tell about yourself
   - Closing: Best wishes, {{your name}}

5. Check spelling:
   - Use spell-check (red underlines)
   - Right-click on mistakes for suggestions

6. Add:
   - Your name in **bold**
   - The date
   - Maybe a simple picture

7. Save your work: "My English Letter"

8. Print or share with teacher

**Skills practiced:**
- Typing in English
- Using spell-check
- Formatting text
- Saving files properly
